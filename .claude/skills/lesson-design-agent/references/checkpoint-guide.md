# 元反思检查点详细指南

## 6 问模板

每阶段结束必须回答以下 6 个问题：

| 序号 | 问题 | 通过标准 |
|------|------|---------|
| 1 | 目标对齐：本阶段产出是否服务于核心教学目标？ | 明确回答"是"，能说明如何服务 |
| 2 | 完整性：是否遗漏关键要素？ | 对照该阶段产出清单逐一确认 |
| 3 | 一致性：是否与前期决策矛盾？ | 未发现矛盾或已解释偏差 |
| 4 | 可执行性：下一阶段能否基于此产出继续？ | 产出物完整，可直接使用 |
| 5 | 专业契合：是否体现专业特色？ | 至少 1 处专业元素 |
| 6 | 优化空间：是否有更优方案？ | 已考虑或确认当前最优 |

## 阶段特定问题

### 需求澄清阶段
- 主题是否贴合学生专业？
- 三维目标是否覆盖？
- 课时能承载多少内容？

### 教案初稿阶段
- 每环节是否有三要素？
- 时间分配是否合理？
- 是否融入专业元素？

### 迭代优化阶段
- 修改是否服务于原始需求？
- 修改要求是否逐一响应？
- 新修改是否影响其他部分？

## 不通过处理流程

```
检查未通过
    │
    ▼
记录问题 → 定位原因 → 返回修正 → 重新检查
    │
    └──── 循环直到通过 ────┘
```
