# 法治教育设计模式

> 法治观念显性化设计模式
> 来源：从思政课教案迭代实践中抽象

---

## 适用条件

- 课程内容涉及规则、权利、义务、法律责任等
- 学生需要明确行为边界和法律后果
- 主题适合从道德说教延伸至法治教育

---

## 模式结构

```
真实案例 → 法条原文 → 行为警示 → 价值内化
```

---

## 关键要素

### 1. 将"法治观念"作为独立教学目标维度

不要将其隐含在"公共参与"或"健全人格"中。在教案目标中单独列出：

```markdown
**法治观念**：通过[具体案例]情境探究，学生分析[各方影响]，
掌握[法律知识]，树立依法自保、敬畏法律的法治意识。
```

### 2. 法治素材三段式呈现

| 阶段 | 内容 | 作用 |
|------|------|------|
| 真实案例 | 与学生生活相关的典型案例 | 建立代入感 |
| 法条原文 | 具体的法律条文（注明出处） | 提供行为边界依据 |
| 行为警示 | 对学生行为的明确指引 | 指导实践选择 |

### 3. 法条引入模板

```markdown
**【法律解读】**
**《[法律名称]》第 [X] 条**
[法条原文]

**【行为警示】**
[结合学生年龄和主题，明确行为边界与法律后果]
```

**示例**（未成年人严重暴力犯罪）：
```markdown
**【法律解读】**
**《刑法》第十七条**
已满 12 周岁不满 14 周岁的人，犯故意杀人、故意伤害罪，致人死亡或者以特别残忍手段致人重伤造成严重残疾，情节恶劣，经最高人民检察院核准追诉的，应当负刑事责任。

**【行为警示】**
已满 12 周岁的未成年人实施严重暴力犯罪，同样可能被追究刑事责任。
法律对严重暴力行为有明确底线，任何人都不能逾越。
```

## 教案片段示例

```markdown
### 知学明理：[环节名称]（X 分钟）

**教师活动**：
1. 播放[与主题相关的真实案例]视频片段
2. 组织学生小组讨论：本案中[各方主体]分别受到哪些影响？
3. 出示[相关法律条文]原文，引导学生认识法律后果
4. 总结：[主题相关行为]不仅伤害他人，行为者自身也将面临法律制裁

**学生活动**：
- 观看视频，记录关键信息
- 小组讨论各方影响
- 诵读法条原文中的关键条款
- 结合自身谈谈法律底线意识

**设计意图**：
通过"真实案例 + 法条原文 + 行为警示"三段式呈现，
让学生认识到[主题相关行为]的法律后果，树立依法自保、敬畏法律的意识。
```

## 验收检查

| 检查项 | 标准 |
|-------|------|
| 目标独立 | "法治观念"是否作为独立教学目标维度？ |
| 法条出现 | 课中是否出现具体法律条文原文？ |
| 三段完整 | 是否形成"案例 → 法条 → 警示"的完整链条？ |
| 行为指引 | 是否对学生行为有明确的法律边界指引？ |
| 年龄适配 | 法条选择是否适合学生年龄和认知水平？ |

---

## 常见误区

| 误区 | 表现 | 修正 |
|------|------|------|
| 法治教育隐性化 | 只播放警示视频，无法条原文 | 引入具体法律条文 |
| 法条堆砌 | 一次引入过多法条，学生无法消化 | 每节课聚焦 1-2 个核心法条 |
| 脱离案例 | 直接宣读法条，缺乏情境支撑 | 先案例、后法条、再警示 |
| 过度恐吓 | 只强调惩罚，忽视依法维权 | 同时强调"依法自保"和"敬畏法律" |

---

## 变体

- **权利保护型**：重点讲解学生享有的合法权益及维权途径（如《未成年人保护法》）
- **义务责任型**：重点讲解学生应遵守的行为规范及违法后果（如《治安管理处罚法》）
- **职业法治型**：结合专业场景讲解职业法律规范（如食品安全法、劳动法等）
