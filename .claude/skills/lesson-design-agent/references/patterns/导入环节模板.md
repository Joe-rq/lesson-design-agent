# 导入环节设计模式

> 收集有效的导入方式，支持快速复用

---

## 一、导入模式分类

| 模式 | 核心元素 | 适用场景 | 时间 |
|------|---------|---------|------|
| 声音导入 | 音频素材 | 情感体验类 | 3-5 分钟 |
| 视频导入 | 影像素材 | 视觉冲击类 | 3-5 分钟 |
| 问题导入 | 核心问题 | 思考讨论类 | 3-5 分钟 |
| 故事导入 | 人物故事 | 情感共鸣类 | 3-5 分钟 |
| 数据导入 | 统计数据 | 认知冲击类 | 3-5 分钟 |
| 冲突导入 | 认知冲突情境 | 概念边界辨析类 | 3-5 分钟 |

## 二、模式详解

### 2.1 声音导入

**设计公式**：
```
播放声音 → 猜测/感受 → 揭示主题
```

**案例：声音里的强国路**
```markdown
教师活动：
1. 播放开国大典原声片段（30秒）
2. 提问："这段声音来自哪里？你听到了什么？"
3. 揭示：这是1949年10月1日开国大典的现场录音

学生活动：
- 聆听音频
- 猜测声音来源
- 分享感受

设计意图：
通过历史声音激发好奇心，让学生身临其境感受历史时刻，
自然引出"声音里的强国路"主题。
```

**素材要求**：
- 时长：30秒-1分钟
- 音质：清晰无杂音
- 特点：有辨识度、有感染力

---

### 2.2 视频导入

**设计公式**：
```
播放视频 → 提取信息 → 引发思考
```

**案例模板**：
```markdown
教师活动：
1. 播放[主题相关]视频片段（X分钟）
2. 提问："视频中最触动你的是什么？"
3. 引出本节课主题

学生活动：
- 观看视频
- 提取关键信息
- 分享感受

设计意图：
通过视觉冲击引发情感共鸣，为后续环节做情感铺垫。
```

---

### 2.3 问题导入

**设计公式**：
```
抛出问题 → 思考/讨论 → 聚焦主题
```

**案例模板**：
```markdown
教师活动：
1. 展示课前调研数据："X%的同学认为..."
2. 抛出核心问题："为什么会出现这样的困惑？"
3. 引出本节课要解决的问题

学生活动：
- 思考问题
- 简短讨论
- 表达观点

设计意图：
从学生真实困惑出发，提高课堂参与度和针对性。
```

---

### 2.4 故事导入

**设计公式**：
```
讲述故事 → 设置悬念 → 引入主题
```

**案例模板**：
```markdown
教师活动：
1. 讲述[人物]的故事片段
2. 在关键处设置悬念："他为什么做出这样的选择？"
3. 引出本节课主题

学生活动：
- 聆听故事
- 思考悬念问题
- 产生好奇心

设计意图：
通过人物故事建立情感连接，激发学生探究欲望。
```

---

### 2.5 数据导入

**设计公式**：
```
展示数据 → 对比分析 → 引发思考
```

**案例模板**：
```markdown
教师活动：
1. 展示一组对比数据（如：中国GDP变化）
2. 提问："从数据中你发现了什么？"
3. 引出本节课主题

学生活动：
- 观察数据
- 分析变化
- 分享发现

设计意图：
通过数据冲击引发认知冲突，激发探究欲望。
```

### 2.6 冲突导入

**设计公式**：
```
呈现冲突情境 → 制造认知失衡 → 聚焦核心概念边界 → 引出总议题
```

**适用主题**：
- 概念边界模糊的主题（如"合作 vs 竞争""礼貌 vs 冒犯"）
- 学生普遍存在认知误区的主题
- 需要辨析价值冲突的主题

**三种具体形式**：

**形式 A：找茬/辨错游戏**
```markdown
教师活动：
1. 展示一组与主题相关的场景图片或短视频（包含边界模糊行为）
2. 提问："这些行为中哪些'不对劲'？是否属于[本课核心概念]？"
3. 不急于给出答案，记录学生分歧
4. 引出总议题："今天我们就来拆解这个边界"

学生活动：
- 观察图片/视频
- 找出"不对劲"行为
- 发表判断并说明理由

设计意图：
通过认知冲突激活学生先验经验，让学生意识到[核心概念边界]并不清晰，
自然产生探究动机，为后续概念学习做铺垫。
```

**形式 B：悖论/反常识情境**
```markdown
教师活动：
1. 抛出一个与主题相关的反常识情境或数据
2. 提问："为什么会出现这种看似矛盾的现象？"
3. 引出本课要解决的认知冲突

学生活动：
- 思考反常识现象
- 尝试解释原因
- 表达初步观点
```

**形式 C：两难判断**
```markdown
教师活动：
1. 呈现一个与主题相关的两难情境
2. 给出 A/B 两个看似都有道理的选项
3. 让学生快速站队并说明理由
4. 引出总议题："面对这种情况，有没有更好的选择？"

学生活动：
- 阅读情境
- 做出初步判断
- 简要陈述理由
```

**关键原则**：
- 冲突点必须**直指本课核心概念边界**，而非泛泛"引起兴趣"
- 冲突情境应来自学生真实生活或专业场景
- 制造冲突后迅速引出总议题，不拖沓、不提前揭晓答案

## 三、导入环节验收

| 检查项 | 标准 |
|-------|------|
| 时长 | ≤ 课时 10%，建议 3-5 分钟 |
| 学生参与 | 至少 1 个学生活动 |
| 主题关联 | 自然引出主题 |
| 情感/认知铺垫 | 为后续环节做铺垫 |
| 冲突制造 | 冲突点是否直指本课核心概念边界（而非泛泛引起兴趣） |
| 核心概念边界 | 导入是否让学生明确意识到本课要解决的认知冲突是什么 |

---

## 四、选择指南

| 主题类型 | 推荐导入 | 理由 |
|---------|---------|------|
| 历史回顾类 | 声音/视频导入 | 身临其境感 |
| 思辨讨论类 | 问题/冲突导入 | 直击困惑、制造认知冲突 |
| 概念边界辨析类 | 冲突导入 | 激活先验经验、聚焦核心问题 |
| 人物故事类 | 故事导入 | 情感共鸣 |
| 社会现象类 | 数据/冲突导入 | 认知冲击 |
