# 导入环节设计模式

> 收集有效的导入方式，支持快速复用

---

## 一、导入模式分类

| 模式 | 核心元素 | 适用场景 | 时间 |
|------|---------|---------|------|
| 声音导入 | 音频素材 | 情感体验类 | 3-5 分钟 |
| 视频导入 | 影像素材 | 视觉冲击类 | 3-5 分钟 |
| 问题导入 | 核心问题 | 思考讨论类 | 2-3 分钟 |
| 故事导入 | 人物故事 | 情感共鸣类 | 4-5 分钟 |
| 数据导入 | 统计数据 | 认知冲击类 | 2-3 分钟 |

---

## 二、模式详解

### 2.1 声音导入

**设计公式**：
```
播放声音 → 猜测/感受 → 揭示主题
```

**案例：声音里的强国路**
```markdown
教师活动：
1. 播放开国大典原声片段（30秒）
2. 提问："这段声音来自哪里？你听到了什么？"
3. 揭示：这是1949年10月1日开国大典的现场录音

学生活动：
- 聆听音频
- 猜测声音来源
- 分享感受

设计意图：
通过历史声音激发好奇心，让学生身临其境感受历史时刻，
自然引出"声音里的强国路"主题。
```

**素材要求**：
- 时长：30秒-1分钟
- 音质：清晰无杂音
- 特点：有辨识度、有感染力

---

### 2.2 视频导入

**设计公式**：
```
播放视频 → 提取信息 → 引发思考
```

**案例模板**：
```markdown
教师活动：
1. 播放[主题相关]视频片段（X分钟）
2. 提问："视频中最触动你的是什么？"
3. 引出本节课主题

学生活动：
- 观看视频
- 提取关键信息
- 分享感受

设计意图：
通过视觉冲击引发情感共鸣，为后续环节做情感铺垫。
```

---

### 2.3 问题导入

**设计公式**：
```
抛出问题 → 思考/讨论 → 聚焦主题
```

**案例模板**：
```markdown
教师活动：
1. 展示课前调研数据："X%的同学认为..."
2. 抛出核心问题："为什么会出现这样的困惑？"
3. 引出本节课要解决的问题

学生活动：
- 思考问题
- 简短讨论
- 表达观点

设计意图：
从学生真实困惑出发，提高课堂参与度和针对性。
```

---

### 2.4 故事导入

**设计公式**：
```
讲述故事 → 设置悬念 → 引入主题
```

**案例模板**：
```markdown
教师活动：
1. 讲述[人物]的故事片段
2. 在关键处设置悬念："他为什么做出这样的选择？"
3. 引出本节课主题

学生活动：
- 聆听故事
- 思考悬念问题
- 产生好奇心

设计意图：
通过人物故事建立情感连接，激发学生探究欲望。
```

---

### 2.5 数据导入

**设计公式**：
```
展示数据 → 对比分析 → 引发思考
```

**案例模板**：
```markdown
教师活动：
1. 展示一组对比数据（如：中国GDP变化）
2. 提问："从数据中你发现了什么？"
3. 引出本节课主题

学生活动：
- 观察数据
- 分析变化
- 分享发现

设计意图：
通过数据冲击引发认知冲突，激发探究欲望。
```

## 三、导入环节验收

| 检查项 | 标准 |
|-------|------|
| 时长 | ≤ 课时 10% |
| 学生参与 | 至少 1 个学生活动 |
| 主题关联 | 自然引出主题 |
| 情感铺垫 | 为后续环节做铺垫 |

---

## 四、选择指南

| 主题类型 | 推荐导入 | 理由 |
|---------|---------|------|
| 历史回顾类 | 声音/视频导入 | 身临其境感 |
| 思辨讨论类 | 问题导入 | 直击困惑 |
| 人物故事类 | 故事导入 | 情感共鸣 |
| 社会现象类 | 数据导入 | 认知冲击 |
