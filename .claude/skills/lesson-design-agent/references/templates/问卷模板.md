# 课前调查问卷模板

> 学情调研阶段使用，收集学生认知起点、情感态度、真实困惑

---

## 一、问卷结构（5 板块）

| 板块 | 问题类型 | 目的 | 建议题数 |
|------|---------|------|---------|
| 基础画像 | 单选/填空 | 了解班级构成 | 2-3 题 |
| 认知起点 | 量表题 | 诊断已有认知水平 | 2-3 题 |
| 情感态度 | 量表题 | 了解情感倾向 | 2-3 题 |
| 真实困惑 | 开放题 | 收集学生疑问 | 1-2 题 |
| 任务收集 | 录音/作品 | 收集课堂展示素材 | 1 题 |

**实体约束**：总问题数 ≤ 15 题

---

## 二、问题模板库

### 2.1 基础画像

```markdown
1. 你的性别是？
   - 男
   - 女

2. 你的政治身份是？
   - 群众
   - 共青团员
   - 入党积极分子

3. 你来自？
   - 城镇
   - 农村
```

### 2.2 认知起点（量表题）

```markdown
4. 你对 [主题] 的了解程度？
   - 非常了解（5分）
   - 比较了解（4分）
   - 一般（3分）
   - 不太了解（2分）
   - 完全不了解（1分）

5. 你能说出 [相关概念] 的含义吗？
   - 能清晰解释
   - 能简单说明
   - 听说过但不理解
   - 完全不知道
```

### 2.3 情感态度（量表题）

```markdown
6. 你认为 "[观点]" 和 "[观点]" 矛盾吗？
   - 完全不矛盾
   - 有点矛盾
   - 比较矛盾
   - 非常矛盾
   - 不确定

7. 你认为学习 [专业] 和 [主题] 有关系吗？
   - 关系密切
   - 有一定关系
   - 关系不大
   - 完全没关系
```

### 2.4 真实困惑（开放题）

```markdown
8. 关于 "[主题]"，你有什么困惑或疑问？
   [文本框，不限字数]

9. 你认为普通人能为 [某事] 做什么？
   [文本框，不限字数]
```

### 2.5 任务收集

```markdown
10. 请录制 30 秒音频：说说你对 [主题] 的理解
    [上传录音文件]

或

10. 请拍摄一张与 [主题] 相关的照片并简述
    [上传图片 + 文字说明]
```

## 三、问卷设计原则

### 3.1 问题设计

| 原则 | 说明 | 示例 |
|------|------|------|
| 具体化 | 避免抽象问题 | ❌ "你爱国吗？" → ✅ "你认为普通人能为国家做什么？" |
| 中性化 | 避免引导性提问 | ❌ "你应该..." → ✅ "你认为..." |
| 可量化 | 使用量表题 | 1-5 分量表，便于统计分析 |

### 3.2 时间控制

| 类型 | 预计时间 |
|------|---------|
| 填空/单选 | 10 秒/题 |
| 量表题 | 15 秒/题 |
| 开放题 | 1-2 分钟/题 |
| 录音/上传 | 2-3 分钟 |

**总时长建议**：5-8 分钟

## 四、数据分析模板

### 4.1 学情分析报告结构

```markdown
# 学情分析报告

## 一、班级画像
- 人数：X 人
- 性别比例：男 X%，女 X%
- 政治身份：团员 X%，群众 X%
- 家庭背景：城镇 X%，农村 X%

## 二、认知起点
- [概念1] 了解程度：均分 X 分
- [概念2] 了解程度：均分 X 分
- 认知薄弱点：[分析]

## 三、情感特征
- 性格分布：外向 X%，内向 X%
- 爱好特长：[分类统计]
- 情感倾向：[分析]

## 四、困惑聚焦
1. [典型困惑1]（X 人提及）
2. [典型困惑2]（X 人提及）
3. [典型困惑3]（X 人提及）

## 五、教学启示
- 针对认知薄弱点：[活动设计建议]
- 针对情感特征：[参与方式建议]
- 针对真实困惑：[讨论环节设计]
```

### 4.2 关键洞察提取

| 数据类型 | 洞察用途 |
|---------|---------|
| 学生困惑 | 课堂讨论素材 |
| 爱好分布 | 分组依据 |
| 性格分布 | 活动形式选择（内向学生需要低压力参与方式） |
| 认知起点 | 教学内容难度调整 |
