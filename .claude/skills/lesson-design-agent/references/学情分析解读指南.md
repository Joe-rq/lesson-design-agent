# 学情分析解读指南

> 将学情调研数据转化为教学设计调整规则
> 当前版本：v1.0

## 学情数据 → 教学调整映射表

| 学情发现 | 调整策略 | 示例 |
|---------|---------|------|
| 历史节点认知率 > 80% | 减少讲解时间，增加活动深度 | 开国大典不讲背景，直接聆听+感悟 |
| 历史节点认知率 < 50% | 增加背景铺垫，使用可视化时间轴 | 课前发放历史大事年表 |
| 职业迷茫率 > 60% | 增加专业关联和职业规划内容 | 增加"期末目标承诺表" |
| 职业目标明确率 > 70% | 强化专业与主题的深层关联 | 用职业精神诠释主题价值 |
| 新媒体依赖度高 | 音视频素材比例 > 60% | 每环节至少1个音频/视频 |
| 传统媒体偏好 | 增加纸质材料和书写环节 | 使用记录卡、手绘时间轴 |
| 课堂偏好：视频 > 讨论 | 控制讨论时长，增加观看素材 | 视频5min+讨论3min |
| 课堂偏好：讨论 > 视频 | 减少视频数量，增加小组研讨 | 讨论5min+视频3min |
| 团员比例低 | 增加榜样代入（非党员榜样） | 用00后运动员而非革命烈士 |
| 团员比例高 | 可引入党内榜样和先进人物 | 用党员先锋、劳模故事 |
| 个别情绪低落 | 避免过度沉重话题，增加温暖环节 | 结尾用集体朗诵而非沉重反思 |
| 班级氛围活跃 | 增加竞争性活动和展示环节 | 小组PK、成果展示墙 |
| 班级氛围内向 | 增加低门槛参与和书写环节 | 个人记录卡、匿名投票 |
| 对主题兴趣低 | 增加游戏化和体验式环节 | 模拟人生、角色盲盒 |
| 对主题兴趣高 | 增加深度思考和延伸讨论 | 价值辨析、现实关联 |

## 问卷板块设计指南

### 5 板块结构（共 15 题）

| 板块 | 题数 | 目的 | 题型建议 |
|------|------|------|---------|
| 基础画像 | 2-3 题 | 了解班级构成 | 选择题 |
| 认知起点 | 3-4 题 | 诊断已有认知 | 判断题+选择题 |
| 情感态度 | 3-4 题 | 了解情感倾向 | 量表题+开放题 |
| 真实困惑 | 2-3 题 | 收集学生疑问 | 开放题 |
| 任务收集 | 1-2 题 | 收集课堂素材 | 开放题 |

### 各板块关键问题示例

**基础画像**：
- 你的性别/年龄/是否团员？
- 你平时通过什么渠道了解新闻？（多选：抖音/微博/B站/电视/报纸/其他）

**认知起点**：
- 你知道以下哪些历史事件？（多选：开国大典/改革开放/建党百年等）
- "工匠精神"指的是什么？（开放/选择）

**情感态度**：
- 谈到"爱国"，你的第一反应是？（1-5分量表：遥远/抽象/无感/认同/强烈认同）
- 你认为所学专业与国家发展有关系吗？（量表题）

**真实困惑**：
- 关于本课主题，你最想弄清楚的一个问题是什么？
- 在学习/生活中，你遇到过哪些与主题相关的困惑？

**任务收集**：
- 请分享一个你了解的与主题相关的人物/故事/新闻
- 如果有机会，你想对历史上的某个人物说些什么？

## 数据分析后的教学设计调整流程

```
1. 统计各题数据 → 识别班级整体特征
2. 标记异常值 → 识别需要特别关注的个体/群体
3. 对照映射表 → 确定调整方向
4. 修改教案 → 在对应环节应用调整策略
5. 记录调整依据 → 便于后续迭代时追溯
```

## 常见问题

### Q1：学情数据与预期不符怎么办？

**策略**：以数据为准，不要强行推进预设设计。
- 如果学生认知起点远低于预期 → 增加铺垫环节
- 如果学生情感态度偏消极 → 增加体验式、游戏化环节
- 如果学生兴趣度很高 → 增加深度讨论和价值辨析

### Q2：个别学生数据异常如何处理？

**策略**：关注但不迁就。
- 在课堂中设计分层任务，让不同层次学生都能参与
- 在情感环节避免过度触发，增加温暖收尾
- 课后可个别沟通，了解真实原因
