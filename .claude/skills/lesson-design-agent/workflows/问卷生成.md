# 问卷生成工作流

> 根据主题和授课对象生成标准化课前问卷。可从主流程调用，也可独立运行。

## 何时运行 / 何时跳过

**运行条件**：
- 用户只需要生成问卷，不需要走完整教案设计流程
- 或从主流程阶段二调用

**跳过条件**：
- 快速模式下使用典型学情模板替代

## 前置条件

- [ ] 主题已确定
- [ ] 授课对象已确定（年级/专业/人数）

## 步骤

### 1. 确认输入

确认主题和授课对象。若有 design_lock.md，直接从中读取。

### 2. 确定问卷维度

调用 `references/学情分析解读指南.md` 确定问卷 5 板块：
1. 基础画像（2-3 题）
2. 认知起点（2-3 题）
3. 情感态度（2-3 题）
4. 真实困惑（1-2 题）
5. 任务收集（1 题）

### 3. 生成问卷

调用 `references/templates/问卷模板.md` 生成问卷内容。

### 4. 实体约束检查

运行 `scripts/entity_check.py` 验证题目总数 ≤ 15。

```bash
python scripts/entity_check.py --items "题目1,题目2,..." --stage "学情调研"
```

超限时：分板块收集，或合并相似题目。

### 5. 输出问卷

输出最终问卷，附带预期数据分析方法说明。

## 独立运行说明

本工作流可脱离主流程独立运行。前提：提供主题和授课对象信息。
