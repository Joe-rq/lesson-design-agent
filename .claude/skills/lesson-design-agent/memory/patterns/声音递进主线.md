# 声音递进主线

> 来源：模拟人生·声音抉择模式（`references/patterns/创新教学形式库.md`）
> 首次应用：v29「郫县豆瓣匠人模拟」

---

## 模式结构

```
角色代入 → 多重关卡(压力递进) → 抉择反思 → 专业关联
```

## 关键设计要素

1. **声音/感官素材匹配每个关卡的抉择**：翻缸声→工期压力→市场诱惑→品质质疑
2. **记录卡让学生写下选择**：低门槛参与，全员可写
3. **压力值可视化**：30%→60%→90%→归零，递进感强
4. **结局揭晓必须与专业实践建立关联**：匠人故事→"坚守品质就是爱国"

## 适用条件

- 需要将抽象概念转化为学生可体验的情感历程
- 课堂有声音/视频素材支撑
- 学生专业可与价值观形成自然关联

## 变体

- 时间旅行抉择版（历史课）
- 职业人生模拟版（职业指导课）
- 道德困境抉择版（思政课）

## 核心收益

表面环节从 10 个压缩到 3+1 个，主线从"历史罗列"变为"声音递进"，学生参与度显著提升。
